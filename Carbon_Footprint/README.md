The **Carbon Footprint Analysis Project** focuses on understanding and analyzing the carbon footprint of individuals/students using collected survey data.  
The project demonstrates the **complete data workflow**: from data collection using **Google Forms**, cleaning with **Excel**, storage in **SQL**, analysis in **Python (Jupyter Notebook)**, and visualization with **Power BI**.  

This project is ideal for educational institutions or organizations interested in **environmental impact assessment** and helps make **data-driven sustainability recommendations**.

- Build **interactive dashboards** using **Power BI** for visualization.

## üîÑ Project Workflow

### 1. Data Collection
- Data was collected through **Google Forms**, where respondents provided information about energy usage, transportation, diet, and waste habits.
- Responses were exported as **CSV files**.

### 2. Data Cleaning
- Excel was used to **remove duplicates, fix missing values**, and **standardize data**.
- Cleaned datasets were saved in `.xlsx` format for analysis.

### 3. Data Storage
- Cleaned datasets were stored in a **MySQL database** for structured querying.
- Tables include `Respondents`, `Energy`, `Transportation`, `Diet`, and `Waste`.

### 4. Data Analysis (Python)
- Python (Jupyter Notebook) was used to:
  - Perform **exploratory data analysis (EDA)**
  - Calculate average carbon footprint per category
  - Identify trends and anomalies

### 5. Visualization (Power BI)
- Built **interactive dashboards** to analyze:
  - Carbon footprint by category
  - High vs low footprint groups
  - Trends across respondents
